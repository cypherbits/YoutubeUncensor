# **YoutubeUncensor**
Java 8/JavaFX (Java8 only) GUI for collecting Youtube videos using the youtube-dl tool.

 - Create a directory called *"bin"* and place inside the *youtube-dl* executables. For Linux/Mac and then for Windows (.exe).
